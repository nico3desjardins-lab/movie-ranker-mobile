# Movie Ranker Mobile

Prototype Next.js mobile-first pour classer des films en famille.

## Déploiement

- Importer ce dépôt dans Vercel
- Framework détecté : Next.js
- Commande de build : `next build`

## Notes

Ce prototype est autonome et n'utilise pas encore Supabase.
La persistance réelle, les affiches, le son et le moteur de duel avancé viendront dans une prochaine étape.
